# Manuscript Draft: Bridge to Human-Subject Pilot Study

## Bridge to Human-Subject Pilot Study

The synthetic framework prepares the project for a human-subject pilot study by clarifying the experimental conditions, task structure, data schema, and primary outcomes. The next empirical phase should recruit learners completing introductory machine-learning reasoning tasks under randomly assigned support conditions. The pilot should test whether adaptive timing-and-fading scaffolds improve transfer and calibration while reducing overreliance compared with unrestricted AI assistance.

The task domain should focus on conceptual ML reasoning rather than advanced implementation. Suitable tasks include selecting an appropriate model, interpreting train-test splits, diagnosing overfitting and underfitting, choosing evaluation metrics, identifying feature leakage, and explaining bias-variance tradeoffs. These tasks are appropriate because they can be varied for transfer while remaining scoreable and instructionally meaningful.

Each learner interaction should record task identity, condition, attempt count, response correctness, confidence, elapsed time, scaffold request behavior, scaffold type, scaffold intensity, policy reason, and follow-up performance. This event structure allows the analysis to distinguish immediate supported success from later independent performance.

The main empirical comparison should be between unrestricted AI assistance and adaptive timing-and-fading scaffolding. If the framework is correct, unrestricted AI may produce strong immediate performance, but adaptive timing-and-fading should produce stronger transfer, lower overreliance, and better calibration. The pilot should therefore treat immediate accuracy as a secondary outcome and prioritize transfer and independent recovery.

The pilot study should also examine mediation pathways. Reduced premature help-seeking and lower high-intensity scaffold use may explain why timing-and-fading improves transfer. Improved calibration may also mediate the relationship between metacognitive scaffolding and independent performance. These pathways would help move the paper beyond a simple condition comparison and toward a theory of how disciplined AI scaffolding supports learning.
